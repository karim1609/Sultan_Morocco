\section{Strategie de tests recommandee}

\begin{enumerate}
  \item \textbf{Feature} : \texttt{GET /api/hotels} structure JSON, pagination bornes.
  \item \textbf{Feature} : auth login/signup/logout + protection routes \texttt{auth}.
  \item \textbf{Feature} : messagerie -- isolation conversations, anti auto-message.
  \item \textbf{Unit} : \texttt{Conversation::findOrCreateDirectBetween}.
  \item \textbf{E2E} (Dusk/Playwright) : parcours visiteur $\rightarrow$ signup $\rightarrow$ message.
\end{enumerate}
