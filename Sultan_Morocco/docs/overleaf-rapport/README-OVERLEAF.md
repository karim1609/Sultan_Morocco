# Rapport SENIOR - Overleaf

## Import

1. Upload `Sultan-Morocco-Rapport-Overleaf.zip` sur Overleaf.
2. **Main document** = **`main.tex`** (PAS main-overleaf.tex).
3. **Compiler** = **pdfLaTeX**.
4. **Recompile** (cliquez 2 fois si besoin pour la table des matieres).

## Contenu (~60+ pages)

- Resume executif + Abstract
- 10 chapitres : strategie, architecture, specs, API, donnees, front, securite, qualite/risques, conclusion
- Diagrammes TikZ (couches, MVC, ERD, deploiement, flux)
- Figures de flux multi-etapes
- Emplacements captures ecran
- Annexes, glossaire, roadmap

## Captures d'ecran (integrees)

Toutes les images sont dans `figures/screenshots/` :

welcome, hotels, restaurants, map, login, signup, profile, messages-inbox, messages-thread, chatbot

Re-uploadez le ZIP pour les inclure sur Overleaf.

## Secours

Si `main.tex` echoue : utilisez `main-overleaf.tex` (version courte).
